  1. What is Linux?
    Linux is an open-source operating system based on UNIX, managed under the GPL license.
  2. Who developed Linux?
    It was developed by Linus Torvalds in 1991.
  3. What is the kernel?
    The kernel is the core part of the OS that interacts directly with hardware.
  4. What is a shell?
    A shell is a command interpreter that lets users interact with the OS using commands.
  5. What is the file system?
    It organizes and stores data in files and directories.
  6. What is meant by open-source?
    Software whose source code is available freely for modification and redistribution.
  7. What is a command in Linux?
    A command is an instruction to perform a specific task (e.g., ls, cd, mkdir).
  8. What is a home directory?
    The default directory assigned to each user in Linux.
  9. Define inode.
    Inode is a data structure containing information about a file.
  10. What is a process?
    A running instance of a program in memory.

Q1. Explain the Architecture of Linux with a Neat Diagram.

Answer: The Linux architecture has 4 layers:

  • 1. Hardware Layer: Physical devices like CPU, memory, I/O.
  • 2. Kernel: Manages resources, device drivers, system calls.
  • 3. Shell: Interface between user and kernel.
  • 4. Application Layer: User-level programs (editors, compilers, browsers).

|  User Applications         |
+----------------------------+
|  Shell / Command Interface |
+----------------------------+
|  Kernel (Device Drivers)   |
+----------------------------+
|  Hardware (CPU, Memory)    |
+----------------------------+

Q2. What are the main features of Linux?

  • Multitasking and multiuser support
  • Portability (works on multiple hardware)
  • Security and stability
  • Open-source and free
  • Networking and flexibility

Q3. Explain important Linux commands with examples.

Command Description Example
ls Lists files and directories ls -l
cd Change directory cd /home
mkdir Create new directory mkdir myfolder
rm Remove file rm file.txt
chmod Change file permissions chmod 755 test.sh

Q4. Explain process management in Linux.

Each process in Linux has a unique process ID (PID). Commands like ps, top, and kill are used to manage processes.

Q5. Explain shell scripting with an example.

A shell script is a file containing Linux commands executed sequentially.

#!/bin/bash
echo "Enter your name: "
read name
echo "Hello, $name! Welcome to Linux."

Save as hello.sh and run using sh hello.sh.
