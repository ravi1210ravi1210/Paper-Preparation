Q1. Explain Android Architecture with Diagram.

Answer: Android architecture consists of 4 layers:

  • 1. Applications Layer: All user apps like Contacts, Camera, etc.
  • 2. Application Framework: Provides APIs for developers (Activity Manager, Content Provider).
  • 3. Libraries & Runtime: SQLite, WebKit, OpenGL + ART (Android Runtime).
  • 4. Linux Kernel: Hardware drivers, memory, power management.

Q2. Explain the main components of Android.

  • Activity: Represents a UI screen.
  • Service: Runs background tasks (e.g., music, updates).
  • Broadcast Receiver: Responds to system-wide events.
  • Content Provider: Manages shared data between apps.
  • Intent: Used for communication between components.

Q4. Explain Intent and its types.

Intent: A messaging object for starting activities or broadcasting.

Types:

  • Explicit Intent: Launches a specific activity.
    Example: 
    Intent i = new Intent(this, SecondActivity.class);
startActivity(i);
  • Implicit Intent: Requests system to perform an action (e.g., open browser).

Q7. Describe Layouts used in Android.

  • Linear Layout: Arranges views in a single direction.
  • Relative Layout: Positions elements relative to others.
  • Constraint Layout: Modern, flexible layout.
  • Frame Layout: Stack elements on top of each other.
