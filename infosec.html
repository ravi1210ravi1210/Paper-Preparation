  1. Define Information Security.
    Protection of information from unauthorized access, use, or destruction.
  2. What is Encryption?
    Process of converting plaintext into ciphertext to protect data.
  3. What is Decryption?
    Converting ciphertext back to readable plaintext.
  4. What is Cryptography?
    Science of securing communication using codes and ciphers.
  5. Define Firewall.
    A security device that filters incoming and outgoing network traffic.
  6. What is an Attack?
    Any action that compromises the security of information.
  7. Define Authentication.
    Verifying the identity of a user or system.
  8. What is Integrity?
    Ensuring data is not altered or tampered with.
  9. Define Virus.
    Malicious code that replicates and damages systems.
  10. What is a Digital Certificate?
    An electronic document that verifies identity using public key infrastructure.

Q1. Explain the CIA Triad in Information Security.

  • Confidentiality: Ensuring information is accessible only to authorized users.
  • Integrity: Maintaining accuracy and completeness of data.
  • Availability: Ensuring information is available when needed.

Q2. Explain Symmetric and Asymmetric Encryption.

  • Symmetric Encryption: Uses same key for encryption and decryption. Example: DES, AES.
  • Asymmetric Encryption: Uses public and private keys. Example: RSA, ECC.

Q3. Explain different types of Security Attacks.

  • Active Attack: Altering or modifying data (e.g., DoS, Spoofing).
  • Passive Attack: Monitoring data transmission (e.g., Sniffing).
  • Phishing: Fake emails to steal credentials.
  • Man-in-the-Middle: Intercepting communication between two parties.

Q4. Explain Firewalls and their types.

  • Packet Filtering Firewall: Filters packets based on IP and ports.
  • Proxy Firewall: Acts as an intermediary between user and network.
  • Stateful Inspection Firewall: Monitors active connections and traffic state.
  • Next-Generation Firewall: Includes deep packet inspection and malware protection.

Q5. Explain Cryptography Techniques.

  • Substitution Cipher: Replaces characters (Caesar cipher).
  • Transposition Cipher: Rearranges character positions.
  • Hashing: Converts data into a fixed-length code (e.g., SHA-256).

Q6. Explain Intrusion Detection System (IDS) and its types.

  • Host-Based IDS (HIDS): Monitors system-level activities.
  • Network-Based IDS (NIDS): Monitors network traffic.

Q10. Explain Access Control Mechanisms.

  • DAC (Discretionary Access Control): Owner decides access.
  • MAC (Mandatory Access Control): System enforces rules.
  • RBAC (Role-Based Access Control): Access based on user roles.
